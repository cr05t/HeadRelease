# 頭痛情報ランディングページ

Headache Information Landing Page のコード一式です。元デザイン: https://www.figma.com/design/TynNkLcpnMQCursz4xaKz2/Headache-Information-Landing-Page

## セットアップ

1. 依存関係をインストール: `npm install`
2. 開発サーバー起動: `npm run dev`

## ローカルでの閲覧

- ブラウザで確認: `http://localhost:5173/`
- 同一LANで共有したい場合: `npm run dev -- --host --port 5173` を起動し、表示される Network アドレスを共有する（例: `http://192.168.x.x:5173/`）
